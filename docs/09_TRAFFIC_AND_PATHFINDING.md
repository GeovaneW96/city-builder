# Traffic and Pathfinding (Phase 2 Overview)

## Purpose

Traffic is a major city-builder system. Phase 2 uses an abstract congestion model rather than per-vehicle agent simulation.

The full specification for the Phase 2 traffic model — including trip generation, road capacity, congestion calculation, testing, and all balance values — lives in `docs/26_TRAFFIC_MODEL.md`.

Phase 3 agent-based traffic, road hierarchy, and intersection logic are specified in `docs/34_AGENT_TRAFFIC_AND_ROAD_HIERARCHY.md`.
