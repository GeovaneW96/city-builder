# Accessibility and Settings

## Purpose

Accessibility should be considered early so the game is easier to use and maintain.

## Initial Accessibility Goals

- readable UI;
- clear icons plus text;
- adjustable volume;
- keyboard shortcuts;
- avoid color-only feedback;
- pause button;
- speed controls;
- tooltips.

## Color and Icons

Warnings should not rely only on color.

Use:

- icon;
- color;
- text;
- tooltip.

## Motion

Later support:

- reduced motion;
- disable camera shake;
- disable excessive animations.

## Text

UI text should be:

- concise;
- readable;
- scalable;
- understandable.

## Controls

Recommended shortcuts:

| Action   | Shortcut |
| -------- | -------- |
| Cancel   | Esc      |
| Pause    | Space    |
| Speed 1  | 1        |
| Speed 2  | 2        |
| Speed 3  | 3        |
| Demolish | D        |
| Roads    | R        |
| Zones    | Z        |

## Settings Menu

Eventually include:

- graphics quality;
- music volume;
- SFX volume;
- camera sensitivity;
- edge scrolling;
- reduced motion;
- autosave interval.
