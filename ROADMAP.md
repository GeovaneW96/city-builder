# Roadmap

ROADMAP phases map to `MASTER_FEATURE_LIST.md` (MFL) phases. See `docs/PLANNING.md` for the full mapping.

| Phase | Goal | MFL Phase |
|---|---|---|
| 1 — Interaction Prototype | Prove building on the map feels good | Scaffolding only |
| 2 — Simulation | Turn the prototype into a game | Phase 1 (Foundation) |
| 3 — Progression & Retention | Make players want to continue | Phase 1 (Foundation) |
| 4 — Depth Systems | Add strategic complexity | Phase 2 (Services & Depth) |
| 5 — Content & Polish | Improve replayability and presentation | Phase 3 + Phase 4 |
